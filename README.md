# C# Student Grade Manager

## Overview
This project is a simple Student Grade Manager written in C#. It allows you to:
- Add student names and scores
- List all students with their letter grades
- Save data to a file
- Load data from a file

## Features
- Uses **classes** and **structs**
- Demonstrates **loops, conditionals, functions**
- Reads and writes student data to a text file
- Calculates letter grades automatically

## Requirements
- .NET 9.0 SDK
- Windows, MacOS, or Linux with console access
